## Features 
#### ✅ - Currently Supported
- See calendar in Day, Week, Month, and Year (WIP) Views
- Navigate to different dates using the side panel calendar
- Add calendar event with different features - Single Event, Whole/Multiday Events, Repeated Events
- Adaptive rendering of overlapping event-blocks or multi-line event-blocks

#### ⚠️ - Work In Progress
- Edit Event Page
- Edit, Delete Functionality
  - Edit based on the type
  - Delete based on the type
- Year Calendar View
- Event Invite Functionality
- Friend Functionality
